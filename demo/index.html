üì• Installation

Install the component library via npm:

npm install @soul/cursor-components

Or use via CDN

<script type="module" src="https://unpkg.com/@soul/cursor-components"></script>

üöÄ Quick Start

HTML

Simply import the library and use the components as custom HTML elements:

<!DOCTYPE html>
<html>
<head>
  <script type="module" src="path/to/soul-cursor-components.js"></script>
</head>
<body>
  <trending-widget></trending-widget>
  <main-navigation active="dashboard"></main-navigation>
</body>
</html>

React

import '@soul/cursor-components';

function App() {
  return (
    <div>
      <trending-widget />
      <main-navigation active="dashboard" />
    </div>
  );
}

Vue

<template>
  <div>
    <trending-widget></trending-widget>
    <main-navigation active="dashboard"></main-navigation>
  </div>
</template>

<script>
import '@soul/cursor-components';

export default {
  name: 'App'
}
</script>
